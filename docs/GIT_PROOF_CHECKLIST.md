# Git Proof Checklist

Use this checklist when reviewing the project assets.

## Local Git proof

- Show this repository in GitHub/Azure Repos.
- Show commit history.
- Show `src/python` scripts.
- Show `artifacts/reports` deliverables.
- Show `artifacts/evidence_images` screenshots.
- Show `docs/AZURE_FABRIC_EXPORT_TO_GIT.md`.
- Explain that large fact data is stored outside Git and tracked by manifest.

## Azure/Fabric proof

- Open Fabric workspace.
- Show Git integration connected to the same repo.
- Show committed workspace items.
- Show Lakehouse tables or Warehouse model.
- Show Data Factory pipeline run history.
- Show Power BI report connected to the semantic model.

## Data honesty note

The product catalog source is public Leatt ZA website data. The transaction/customer rows are synthetic and are used for demonstration, performance testing, BI modeling, and ML validation.
